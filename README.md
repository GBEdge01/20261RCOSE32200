# Python Practice Programs

파이썬 기초 문법을 연습하기 위한 간단한 콘솔 프로그램 모음입니다. 사칙연산, 별 찍기, 계산 게임, 전투 게임 예제가 포함되어 있습니다.

## 실행 환경

- Python 3.x
- 별도 설치가 필요한 외부 라이브러리는 없습니다.

## 파일 구성

| 파일명 | 설명 |
| --- | --- |
| `sum.py` | 두 숫자를 입력받아 더한 값을 출력합니다. |
| `sub.py` | 두 숫자를 입력받아 뺀 값을 출력합니다. |
| `mul.py` | 두 숫자를 입력받아 곱한 값을 출력합니다. |
| `div.py` | 두 숫자를 입력받아 나눈 값을 출력합니다. |
| `mod.py` | 두 숫자를 입력받아 나머지를 출력합니다. |
| `star.py` | 입력한 줄 수와 별 개수에 따라 사각형 형태로 별을 출력합니다. |
| `star_boundary_X.py` | 입력한 크기에 따라 테두리와 X 모양이 있는 별 패턴을 출력합니다. |
| `christmas_tree.py` | 입력한 높이에 따라 트리 모양의 별 패턴을 출력합니다. |
| `calculate_game.py` | 제한 시간 안에 덧셈 문제를 푸는 계산 게임입니다. |
| `fighting_game.py` | 공격과 방어를 선택하며 적과 싸우는 간단한 전투 게임입니다. |

## 실행 방법

터미널에서 원하는 파일을 선택해 실행합니다.

```bash
python 파일명.py
```

예시:

```bash
python calculate_game.py
python fighting_game.py
python christmas_tree.py
```

## 프로그램 설명

### 계산 게임

`calculate_game.py`는 총 5개의 덧셈 문제를 출제합니다. 문제마다 제한 시간을 설정할 수 있으며, 제한 시간 안에 정답을 입력하면 점수를 얻습니다.

### 전투 게임

`fighting_game.py`는 플레이어와 적이 번갈아 행동하는 콘솔 게임입니다.

- `A`: 공격
- `D`: 방어

적의 체력이 0 이하가 되면 승리합니다. 플레이어의 체력이 0 이하가 되면 게임이 종료됩니다.

### 별 찍기

`star.py`, `star_boundary_X.py`, `christmas_tree.py`는 입력한 숫자에 따라 다양한 별 모양을 출력하는 연습 프로그램입니다.

## 참고

일부 파일의 한글 입력 문구가 터미널 인코딩 설정에 따라 깨져 보일 수 있습니다. 이 경우 터미널의 문자 인코딩을 UTF-8로 맞추면 더 잘 표시될 수 있습니다.
